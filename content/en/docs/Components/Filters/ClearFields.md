## Filter *ClearFields*

### Overview
Clear a set of fields of all records passing through

### Configuration

Keys available in the `[filter.config]` section:

|Name|Type|Default|Required|Description|
|:--:|:--:|:-----:|:------:|:---------:|
| Fields| array of strings| | true| set of fields to clear|
