{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} Baker can convert, process, extract or store records (structured data), applying whatever transformation between input and output through easy-to-write filters. It is fully parallel and maximizes usage of both CPU-bound and I/O bound pipelines. Baker is an open source project by NextRoll, inc. and it's completely written in Go {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fas fa-tools" title="Fully customizable" %}} Baker is a Go library, import it in your project, use provided components (or create your own) and run the topology! {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" %}} We encourage all users to contribute to the project by adding new components, improve the core or the documentation. Visit the project on GitHub and feel free to open a new [Pull Request](https://github.com/AdRoll/baker/pulls). {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-bug" title="I found a bug!" %}} If you think you found a bug, please create a [new issue](https://github.com/AdRoll/baker/issues/new) on GitHub {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section >}}
This is a simple example of Baker usage. Check out more examples in the project repository
{{< highlight go >}} package main import ( "log" "strings" "time" "github.com/AdRoll/baker" "github.com/AdRoll/baker/input" "github.com/AdRoll/baker/output" ) // Some example fields const ( Timestamp baker.FieldIndex = 0 Source baker.FieldIndex = 1 Target baker.FieldIndex = 2 ) var fields = map[string]baker.FieldIndex{ "timestamp": Timestamp, "source": Source, "target": Target, } func fieldByName(key string) (baker.FieldIndex, bool) { idx, ok := fields[key] return idx, ok } func main() { toml := ` [input] name = "List" [input.config] files=["testdata/input.csv.zst"] [output] name = "FileWriter" procs=1 [output.config] PathString="./_out/output.csv.gz" ` c := baker.Components{ Inputs: input.All, Outputs: output.All, FieldByName: fieldByName, } cfg, err := baker.NewConfigFromToml(strings.NewReader(toml), c) if err != nil { log.Fatal(err) } var duration time.Duration err = baker.Main(cfg, duration) if err != nil { log.Fatal(err) } } {{< / highlight >}}
